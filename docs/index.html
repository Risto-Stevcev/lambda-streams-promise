LambdaStreamsPromise

Javascript promise helpers for lambda-streams.

Usage

  • Promise.first_to_promise converts the first signal of a finite async stream into a promise.
  • Promise.last_to_promise converts the last signal of a finite async stream into a promise.
  • Promise.from_promise converts a promise into a finite async stream.

See Promise for more info.
